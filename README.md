# PocketMini - 後で読むアプリ

## プロジェクト概要
- **名前**: PocketMini
- **目的**: Pocket類似の「後で読む・タグで整理」に特化したローカル専用MVP
- **特徴**: 
  - **完全オフライン動作**（同期・ログイン不要）
  - **ブックマークレット対応**（ブラウザ拡張不要の保存導線）
  - **最小ファイル構成**（3ファイルのみ）

## 実装済み機能 ✅
1. **URL保存**
   - フォーム入力による保存
   - ブックマークレット経由の自動保存（`?add=URL&title=TITLE`）

5. **データ管理**
   - JSON形式でのエクスポート/インポート
   - 重複URL時は上書き（創作日時更新）

6. **UI/UX**

## データ構造
- **保存場所**: `localStorage["pocket-mini"]`
- **データモデル**: 
```json
{
  "bookmarks": [
    {
      "id": "unique-id",
      "url": "https://example.com",
      "title": "記事タイトル",
      "tags": ["tag1", "tag2"],
      "createdAt": "2025-08-16T11:50:00.000Z",
      "archived": false
    }
  ],
  "lastExportAt": "2025-08-16T11:50:00.000Z"
}
```

## ユーザーガイド
### 基本的な使い方
1. **保存**: URLとタイトルを入力して「保存」ボタン
2. **ブックマークレット**: 「ブックマークレット」ボタンをクリックして表示される青いボタンをブラウザのブックマークバーにドラッグ
3. **閲覧**: 任意のサイトでブックマークレットを実行→アプリが開き自動保存
4. **検索**: 検索バーでタイトル・URL・タグを検索
5. **整理**: タグでの絞り込み、アーカイブ機能で整理
6. **バックアップ**: エクスポート/インポートでデータ管理

### キーボードショートカット
- `Ctrl+S` (Cmd+S): ブックマーク保存
- `ESC`: ブックマークレット表示を閉じる
